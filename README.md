# Project 7: Philly Restaurant Chatbot (Due Sorelle)

## Live Demo
**[philly-restaurant-chatbot.onrender.com](https://philly-restaurant-chatbot.onrender.com)**

## Service Demonstrated
**Custom AI Chatbot for Restaurants** — a chatbot trained on a specific restaurant's
menu, hours, reservations, and FAQs. Handles common customer questions instantly,
captures reservation requests, and reduces front-of-house phone volume.

## Goal
Build a fully functional chatbot demo for a fictional Philadelphia restaurant
("Due Sorelle") — representing the kind of custom chatbot deployed for real
restaurant clients in the Philly area.

## Stack
- Python / Flask
- OpenRouter API (free tier)
- Render (free tier deployment)

## Run Locally
```bash
pip install -r requirements.txt
OPENROUTER_API_KEY=your_key python app.py
```
